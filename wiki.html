Town Management

Commands for running your town effectively.

Command Description
/town deposit <amount> Add money to town bank
/town withdraw <amount> Withdraw money from town bank
/town setspawn Set town spawn point
/town taxes <amount> Set daily resident tax

Town Permissions

Control what different ranks can do in your town.

Command Description
/town rank add <rank> <player> Assign a rank to a resident
/town perm set <rank> <permission> <true/false> Set permissions for a rank
/town perm list List all available permissions

Town Upgrades

Improve your town with these upgrades:

Upgrade Effect Cost
Extra Claims +5 claim blocks 10,000
Increased Mob Caps +10% mob spawns 15,000
Tax Bonus +10% tax income 20,000

Purchase upgrades with: /town upgrade <upgrade>

Nation Basics

Nations are alliances of towns that work together for mutual benefit and protection.

Requirements to form a nation:
  • Must be mayor of a town
  • Your town must be at least level 3
  • 50,000 in-game currency

Creating a Nation

Command Description
/nation create <name> Form a new nation
/nation invite <town> Invite a town to join
/nation join <nation> Accept a nation invitation

Nation Management

Command Description
/nation deposit <amount> Add funds to nation bank
/nation withdraw <amount> Withdraw funds from nation bank
/nation setcapital <town> Set the nation's capital
/nation taxes <amount> Set tax rate for member towns

Alliances & Wars

Command Description
/nation ally <nation> Propose an alliance
/nation enemy <nation> Declare another nation as enemy
/nation war <nation> Declare war on another nation
/nation peace <nation> Propose peace treaty

Making Money

There are several ways to earn in-game currency:

  • Selling items to the server shop (/shop)
  • Selling items to other players
  • Completing jobs (/jobs browse)
  • Participating in events
  • Voting daily (/vote)

Player Shops

Create your own shop to sell items to other players:

Command Description
/shop create Start shop creation
/shop price <amount> Set item price
/shop stock <amount> Set initial stock

Jobs System

Earn money by performing various tasks around the server.

Command Description
/jobs browse View available jobs
/jobs join <job> Start a job
/jobs leave Quit current job
/jobs stats View your job progress

Auctions

Sell items to the highest bidder through the auction house.

Command Description
/ah Open auction house
/ah sell <price> Sell held item
/ah bid <amount> Bid on an auction

PvP Rules

Basic PvP guidelines:

  • PvP is enabled in wilderness areas
  • No PvP in spawn or protected towns
  • Combat logging is prohibited
  • No spawn killing

Raiding

Raiding allows you to steal from unprotected chests in enemy territory.

Rules:
  • Can only raid towns you're at war with
  • Cannot raid capital towns
  • Must declare raid with /raid declare <town>
  • Raids last 30 minutes

Town Sieges

Sieges allow you to capture enemy towns and claim them for your nation.

Command Description
/siege declare <town> Start a siege
/siege join Join ongoing siege
/siege status Check siege progress

Nation Wars

Full-scale conflicts between nations with territory at stake.

War Mechanics:
  • Must be declared by nation leader
  • Lasts 3 real-world days
  • Winning nation can claim one town from loser
  • War score based on kills, objectives, and participation
Command Description
/war declare <nation> Declare war
/war status Check war progress
/war objectives View current war objectives
